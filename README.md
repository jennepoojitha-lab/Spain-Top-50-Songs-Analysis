🎵 Spain Top 50 Songs Analysis Dashboard

This project is a Streamlit-based dashboard that analyzes the Spain Top 50 Songs dataset. It provides interactive insights into song popularity, top artists, explicit vs clean songs, album types, playlist positions, and song duration using Python, Pandas, and Plotly.

Technologies

- Python
- Streamlit
- Pandas
- Plotly

Run

pip install -r requirements.txt
streamlit run app.py
